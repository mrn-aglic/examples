# Dataset description

The dataset was obtained from the following link:
https://www.kaggle.com/datasets/rounakbanik/the-movies-dataset

A subset of the columns from the ratings_small.csv file were
selected. Also a subset of the columns from the movies_metadata.csv
was selected and then the nan values were dropped. 

- movies_metadata.csv -> movies.csv
- rankings_small.csv -> rankings_smaller.csv
